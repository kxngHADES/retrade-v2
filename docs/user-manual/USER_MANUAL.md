## 1. Getting Started

Welcome to ReTrade a C2C platform, before you begin grab a drink Just cause you can

### System Requirements
1. A mobile phone
2. A RICA simcard
3. working email address


### Creating an Account
> [!Note]
> You will need a valid south african phone number to create an account. If you do not have one well that is unfortunate you will need to go to a shop buy a sim card and RICA it

#### Step 1: Go to the Registration Page

* Open ReTrade website or mobile PWA
* Click the `Register` button 
> Usually found in the top right, left navbar or login screen

#### Step 2: Fill in your information
* You will see a registration form
* Enter the required information:
	 * First name
	 * Last Name
	 * Email
	 * Phone Number
	 * Password
	 * Confirm Password
* Double check you phone number is correct it is essential for recieving the OTP

#### Step 3: Submit Form
* Click the `Register` button
* The system will temporarily store your information and send a varification code (OTP) to the phone number you provided.

#### Step 4: Enter the OTP
* Check you mobile phone for a text message containing a numeric code (OTP)
* On the screen, ypu will be taken to an OTP entry page
* Type the code exactly as you recieved it

#### Step 5: Complete Registration
* Click `Verify`
* The system validates your OTP, if the code is correct:
	 * Your account information will be permanently saved
	 * You will automatically be logged in
* You are then redirected to the main page of ReTrade


### What to do if you did not recieve the OTP
* Wait a few seconds and check your phone signal
* Make sure you entered the correct phone number.
* Look for an option to "Resend OTP" on the verification screen
